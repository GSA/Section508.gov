<p>In Module: 14 of the Accessible Name & Description Inspector (ANDI) Tool video series, learn how to use the <a href="https://developer.paciellogroup.com/resources/contrastanalyser/">Color Contrast Analyzer's (CCA)</a> built-in color sampler to ensure the contrast ratio between text, and images of text, and it&apos;s background is sufficient. CCA can be used to when contrast cannot <a href="{{site.baseurl}}/tools/glossary/#programmatically-determinable" >programmatically determined</a> by ANDI.</p>

<video controls="controls" poster="{{site.baseurl}}/assets/images/thumbnails/training-video-andi-poster.jpg" data-vscid="3qesx4ovd" style="width:100%"><source src="https://assets.section508.gov/assets/videos/training-video-andi-14-oc.mp4" type="video/mp4"></video>

  <h4 class="usa-accordion__heading">

  <div id="a1" class="usa-accordion__content">
    <p>Voice Over (VO): Welcome to the Color Contrast Analyzer CCA Module. The purpose of this module is to ensure the contrast ratio between text and images of text and its background is sufficient.</p>
    <p>VO: People who are colorblind or have low vision will only have comparable access if there is sufficient contrast between the text and the background. The color contrast standards are 4.5 to 1 for standard text and 3 to 1 for large text.</p>
    <p>VO: Incidental text and local types are excluded from this requirement. Note, if the text is black on a white background or close to it, there is no need to perform this test.</p>
    <p>VO: To start, open a web page that contains text elements, then open the CCA tool. Select the foreground color picker button and move the color picker box over the darkest part of the text and select enter.</p>
    <p>VO: Next, select the background color picker button and move the color picker box over the background section of the text and select enter. After the foreground and background colors have been identified, the contrast ratio displays in the result luminosity section in the result memo text area box.</p>
    <p>VO: Note, do not select the drop-down menu of the color picker. An element passes when the color contrast ratio meets the minimum contrast requirement of 4.5 to 1 for standard text and 3 to 1 for large text.</p>
    <p>VO: An element fails when the color contrast ratio does not meet the minimum contrast requirement of 4.5 to 1 for standard text and 3 to 1 for large text. The test does not apply when there is no text on the page.</p>
    <p>VO: This concludes the color contrast analyzer CCA module. To download the latest version of the CCA tool, go to http://www.pacillogroup.com/resources/contrastanalyzer.</p>
  </div>
